## 개요
지능형 오디오 방송 단말기로서 음원 재생, 마이크 실시간 방송, 긴급 방송, 외부 센서 연동 및 단독 운영(서버리스)을 지원한다. 실내·실외 환경 모두에 설치 가능하며, 유선/무선 네트워크 기반으로 중앙 관리 혹은 독립 운영이 가능하다.

## 주요 기능
### 1. 설치 및 설정
- 간편 초기 설정(스마트폰·웹 기반)
- 자동 네트워크 검색(mDNS/Zeroconf)
- 장비 자동 등록 및 그룹 관리

### 2. 네트워크 및 연결성
- 유선: Ethernet / PoE(옵션)
- 무선: WiFi, BLE
- 확장 옵션: LTE, LoRa, UHF
- 프로토콜: HTTP/HTTPS, WebSocket, RTSP/RTP, MQTT(선택)

### 3. 방송 기능
#### 3-1. 방송 소스
- 마이크 실시간 방송
- Line-In 입력 방송
- MP3 음원 재생(내부/외부 USB)
- TTS 방송(로컬/클라우드)
- 블루투스 오디오 입력(A2DP)
- 예약 방송(스케줄링)
- 긴급 방송(즉시 우선 재생)

#### 3-2. 방송 제어 & 우선순위
- 방송 우선순위 레벨 설정
- 긴급 방송 중단 → 종료 후 기존 방송 자동 복귀
- 특정 스피커 또는 스피커 그룹 대상 방송

#### 3-3. 방송 동기화
- 다중 스피커 동시 방송 지연 보정(AV Sync)
- 네트워크 지연 상황 자동 버퍼 조정

### 4. 오디오 품질
- AGC(자동 게인 조절)
- Noise Reduction
- Echo Cancellation
- EQ(이퀄라이저) 설정
- 스피커 출력 레벨 제어

### 5. 외부 장치 및 센서 연동
#### 5-1. 센서
- 움직임 감지(사람/동물)
- 연기 감지
- 소리 감지
- 온도·습도 센서

#### 5-2. 제어
- 경광등 ON/OFF 제어
- 릴레이 접점 제어(AC/DC)
- USB 인터페이스
- 직렬 통신: RS-232 / RS-422 / RS-485
- CAN 인터페이스

### 6. 관리 및 모니터링 기능
#### 6-1. 장비 상태 모니터링
- CPU/메모리 사용량
- 온도/과열 감지
- 네트워크 상태(RSSI, 링크 품질)
- 전원 상태/UPS 연동
- 스토리지 사용량
- 센서 상태 표시

#### 6-2. 로그 및 감사
- 방송 기록(시간/종류/발신자)
- 긴급 방송 로그
- 장비 이벤트 로그
- API 호출 로그

#### 6-3. 장애 복구
- 네트워크 장애 자동 복구
- Watchdog 기반 자가 재부팅
- 설정 자동 백업/복원

### 7. 외부 시스템 연동
#### 7-1. REST API
- 단말 조회
- 방송 시작/중지
- 예약 등록/조회
- 장비 설정 관리

#### 7-2. Webhook 지원(센서 이벤트 → 외부 서버로 알림)

### 8. 운영 환경
- 서버리스 단독 운영 가능
- 중앙 서버/클라우드와 연동한 대규모 관리도 가능

### 9. 펌웨어/보안
- OTA 업데이트(Rollback 지원)
- 펌웨어 무결성 검증(서명)
- TLS 통신 암호화
- 관리자 계정/권한 관리
- API 인증(JWT 기반)

### 10. 관리 콘솔
- 웹 기반 관리 콘솔
- 모바일 콘솔
- 내장형 어플라이언스 콘솔(UI/Display 선택)
- 실시간 상태/알림 표시
