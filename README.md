## 개요
IoT 기반의 스마트 방송·알림 시스템을 개발하기 위해 요구사항 분석 중 입니다.

## 주요 기능
- 쉬운 설치 및 설정
- 다양한 네트워크 지원 : 유선(이더넷), 무선(WiFi, BLE, LTE, UHF, LoRa...)
+ 다양한 방송 모드
  - 마이크 방송
  - Line-In 방송
  - 음원(MP3) 방송 : 녹음 / 음원
  - 문안(TTS) 방송
  - 블루투스 방송
  - 예약 방송
  - 긴급 방송
- 방송 우선순위 설정
+ 방송 중 긴급 방송 지원
  - 긴급 방송 후 이전 방송 계속 방송
+ 외부 시스템 연동
  - REST-API 지원 : 스피커 조회 / 방송 / 예약
+ 다양한 센서 연동
  - 사람 / 동물 움직임 감지
  - 연기 감지
  - 소리 감지
  - 온 / 습도 측정
- 경광등 ON / OFF
- 릴레이 접점 제어 : AC / DC
- 서버리스 운영 ⇒ 스피커 단독 운영
+ 다양한 스피커 지원
  - 실외 대형 스피커
  - 실내 소형 스피커
+ 스피커 외부 인터페이스 지원
  - USB
  - RS-232 / RS-422 / RS-485
  - CAN
+ 다양한 콘솔 지원
  - 웹 콘솔
  - 어플라이언스 콘솔
  - 모바일 콘솔
- 스피커 상태 콘솔 표시
